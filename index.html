Newsfeed Eradicator for Safari.

Newsfeed eradicator blocks the distracting Facebook newsfeed. Newsfeed eradicator for Chrome was originally created by Jordwest.

I appreciated the Chrome extension a lot, but I use safari for it's power management.

Update

I don't have a safari developer account anymore so I will not update the extension.

Please check out https://github.com/maxfriedrich/quiet-facebook for a simpler solution.
